### String Methods in Action ğŸ¬


### More Useful String Methods ğŸ› ï¸


### String Formatting Magic âœ¨

Instead of messy concatenation, use f-strings (formatted strings):

```python
name = "Sarah"
age = 25
city = "Boston"

# Old way (works but messy):
message = "Hi, I'm " + name + " and I'm " + str(age) + " years old from " + city

# New way (clean and readable):
message = f"Hi, I'm {name} and I'm {age} years old from {city}"
print(message)

# You can even do calculations inside f-strings!
price = 19.99
print(f"Total with tax: ${price * 1.08:.2f}")  # Rounds to 2 decimal places
```

## Advanced String Techniques ğŸ¯

```python
# String slicing (getting parts of strings)
phrase = "Python Programming"
print(phrase[0])        # P (first character)
print(phrase[-1])       # g (last character)
print(phrase[0:6])      # Python (characters 0 to 5)
print(phrase[7:])       # Programming (from position 7 to end)

# Split strings into lists
sentence = "I love Python programming"
words = sentence.split()  # ['I', 'love', 'Python', 'programming']
print(f"Word count: {len(words)}")

# Join lists back into strings
new_sentence = " ".join(words)  # Put words back together
print(new_sentence)
```

## Working with User Input Like a Pro ğŸ­

```python
def get_clean_input(prompt):
    """Get input from user and clean it up"""
    return input(prompt).strip().title()

def get_number_input(prompt):
    """Get a number from user with error handling"""
    while True:
        try:
            return float(input(prompt))
        except ValueError:
            print("Please enter a valid number!")

# Use these functions for better user experience
name = get_clean_input("Enter your name: ")
age = int(get_number_input("Enter your age: "))
height = get_number_input("Enter your height in feet: ")

print(f"Hello {name}! You are {age} years old and {height} feet tall.")
```
